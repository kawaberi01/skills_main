# Template

断層競馬の参考欄は、必ず次の形で出すこと。

```text
#### ■ レース情報
（例：川崎 7R 姫百合賞）

#### ■ 判定
（作業種類）

#### ■ 求められているもの
（例：意思決定用の概算 / 実装用の詳細など）

#### ■ 粒度
（粗い / 中 / 詳細）

#### ■ 今やること
（最大3つ）

#### ■ 今やらないこと
（重要：必ず書く）

#### ■ 候補手法・フレームワーク
（2〜3個）

#### ■ 次の一手
（1つだけ）

### ■ 思考コメント（任意）
（例：少し詳細に入りかけています / まず分類から始めると良いです）
```

## 書き方ルール

- `判定` は 1 つだけ
- `粒度` は 1 つだけ
- `今やること` は最大 3 つ
- `今やらないこと` は必ず書く
- `候補手法・フレームワーク` は 2 〜 3 個
- `次の一手` は 1 つだけ

## 南関レース向けの言い換え例

### 判定

- `調査`
- `設計`
- `実装`

### 求められているもの

- `予想本体の補助となる参考整理`
- `軸候補と相手候補を分ける前段整理`
- `買い方に入る前の短い方向付け`

### 今やること

- `馬場一致の有無を確認する`
- `人気と条件一致のズレを見る`
- `軸向きか相手向きかを切り分ける`

### 今やらないこと

- `この欄だけで本命を決め直さない`
- `買い目を広げすぎない`
- `APIにない気配情報を主根拠にしない`

### 候補手法・フレームワーク

- `馬場一致`
- `場・距離一致`
- `人気と条件のズレ`
- `騎手×厩舎一致`
- `当日傾向との接続`

## 追加ルール

- 断層競馬の参考欄は、予想本体の後ろに別枠で出す
- 本体の結論と違う観点が出ても、参考欄では「補助観点」として書く
- 長文化しない
