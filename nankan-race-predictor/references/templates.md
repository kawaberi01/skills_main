# 出力テンプレート

出力フォーマットのぶれを減らすため、各モードでこの形を使うこと。  
見出し名と順番はできるだけ固定する。

## 総合買い目型

- オッズ: yes/no

- 反映度合い: 参考程度 / 弱めに反映 / 中程度に反映
- 傾向の要点: ...

熱さ判定
- 本命信頼度: 高 / 中 / 低

- 低配当判定: 妙味あり / 通常 / 的中寄りだが回収薄い

- 相手候補 ...（単勝オッズ: ... / ...）

- 低配当時の扱い: 厚張りしない / 点数を絞る / 軽く遊ぶに落とす
